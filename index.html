Case Studies (Click to Expand)

1. Starlink Shared IP Piracy Investigation

Problem: Unauthorized BitTorrent activity detected on a shared Starlink NAT public IP used by multiple devices.

Diagnosis: Identified NAT limitation preventing direct IP-to-device attribution. Correlated DHCP logs, router connection tables, and time-based traffic patterns.

Solution: Performed structured network forensics using MAC mapping and session timing analysis.

Result: Narrowed down probable source devices and improved investigation process.

2. Phishing Credential Harvesting Detection

Result: Prevented credential compromise.
